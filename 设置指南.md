# GitHub 个人主页创建指南

## 1. 创建专用仓库

1. 登录 GitHub
2. 新建仓库，仓库名称填写你的 **GitHub 用户名**
   - 例如：如果用户名是 `calixlin`，仓库名就是 `calixlin`
3. 仓库设为 **Public**
4. ✅ 勾选 "Add a README file"

## 2. 替换内容

把 `README.md` 中的内容复制进去，重点修改：

```markdown
# 把这些替换成你的用户名
- 用户名: your-username → 改成你的 GitHub 用户名
- 邮箱: 1553181496@qq.com → 保持不变
```

## 3. 常用 Badges

| Badge | 链接 |
|-------|------|
| Profile Views | https://komarev.com/ghpvc/ |
| GitHub Stats | https://github-readme-stats.vercel.app/ |
| Streak | https://github-readme-streak-stats.herokuapp.com/ |

## 4. 快速获取你的数据

- GitHub 主页 → 点击你的头像查看
- 或访问: `https://github.com/your-username?tab=overview`
